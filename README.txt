In order to run the app, do the following:

1. Open the command terminal (cmd.exe)

2. Change the directory to where you downloaded the files (...\F1 Weather App)

3. Enter the following command: npm install

4. After npm has installed, enter the following command: npm start

5. The app should load onto your browser, if it does not then go to your browser and enter the following into the address bar: http://localhost:3000/
